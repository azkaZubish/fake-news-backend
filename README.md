# Backend for Fake News Detection
It uses Machine learning classification algorithms to classify the processed article into real, fake or uncertain.

## Backend Setup

### Navigate to backend folder:
```bash
cd fake_news_backend
```
### Install Dependencies
```bash
pip install -r requirements.txt
```
### Run Backend
```bash
python app.py
```
### Backend runs on:
```bash
http://localhost:5000
```
